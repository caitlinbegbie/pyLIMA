Examples

Examples are available in the pyLIMA source, see ref:`Installation`

-----------------

To run an example, you can use `Juptyer notebook <https://jupyter.org>`_


Click on examples folder :

You can also run (and edit) classic python file :

pyLIMA_example_*.py
